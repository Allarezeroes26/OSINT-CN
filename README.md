# Chinese OSINT Investigation Simulation (CTF) Hacktoria

**Type:** OSINT / Cybersecurity CTF  
**Format:** 10-stage chained investigation  
**Status:** Completed

---

## Overview

A structured OSINT simulation replicating a real-world intelligence workflow across Chinese-language platforms. The challenge focuses on progressive identity reconstruction through staged data correlation and analysis.

Starting from fragmented online traces, the investigation builds toward a full multi-platform profile using cross-referenced public information.

---

## Investigation Flow

- Source discovery in Chinese-language forum data  
- Extraction of contact information from comment sections  
- QQ-based cross-platform identity pivoting  
- Behavioral analysis on Chinese Q&A platforms  
- Professional and location inference from public activity  
- Simulated infrastructure-based data correlation  
- Mobile-to-social platform linking (Weibo / WeChat)  
- Multi-source location verification and triangulation
